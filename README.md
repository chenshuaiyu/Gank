# Gank 客户端

### 简介

这是一款分享干货的APP，此项目使用[Gank.io](http://gank.io/)网站提供的API，基于`MVVM` + `Butterknife` + `Rxjava` + `Retrofit` + `Room` + `Glide` 等架构实现。
